### **Step 3: Referral Income Settings** (`/admin/referral-income-settings`)
1. Click "Referral Income Setting" in sidebar
2. Enable "Multi-Level Income"
3. Configure level percentages (should already be set as defaults):
   - Level 1: 15%
   - Level 2: 10%
   - Level 3: 8%
   - Level 4: 10%
   - Level 5: 6%
   - Level 6-7: 5%
   - Level 8-9: 4%
   - Level 10: 3%
   - Level 11-25: 2%
4. Verify "Total Level Income" shows 100%
5. Enable "Qualification Rules":
   - Level 1-3: 5 Directs
   - Level 4-13: 1 Direct per 2 Levels
   - Level 14-25: 1 Direct per 3 Levels
6. Click "Save Referral Income Settings"

### **Step 4: Payout Settings** (`/admin/payout-settings`)
1. Click "Payout Setting" in sidebar
2. Enable "Weekly Payouts"
3. Set Cutoff: Friday at 23:59
4. Set Release: Monday at 09:00
5. Set Admin Charges: 10%
6. Check all income types for admin charges
7. Click "Save Payout Settings"

### **Step 5: Program Settings** (`/admin/program-settings`)
1. Click "Program Setting" in sidebar
2. Verify Investor Cap: 2.0
3. Verify Leader Cap: 3.0
4. Set Leader Base Amount (e.g., 1000)
5. Click "Save Program Settings"

### **Step 6: Renewal Settings** (`/admin/renewal-settings`)
1. Click "Renewal Setting" in sidebar
2. Enable "ID Renewal Rule"
3. Configure renewal methods
4. Click "Save Renewal Settings"
